函数定义的方式有两种:一种是函数声明,另一种是函数表达式.


            //函数声明格式:function关键字 + 函数名
            function functionName(arg0, arg1, arg2) {
                //函数体
            }

Firefox、Safari、Chrome和 Opera 都给函数定义了一个非标准的 name 属性,通过这个属性可以访问到给函数指定的名字。关于函数式声明,它一个重要特性就是函数声明提升(function declaration hoisting),意思是执行代码之前会先读取函数式声明.这就意味着可以把函数式声明放在调用它的语句后面.这是函数式声明与函数表达式的是唯一区别!

记忆技巧: 有等号的才叫表达式.

FunctionNameExample01.html

            //only work in Firefox, Safari, Chrome和Opera.
            alert(functionName.name); //"functionName"

            sayHi();
            function sayHi() {
                alert("Hi");
            }

第二种创建函数的方式是使用函数表达式.函数表达式有几种不同的语法形式,下面是最常见的形式:


            var functionName = function(arg0, arg1, arg2) {
                //函数体
            };

            sayHi(); //错误,函数还不存在
            var sayHi = function() {
                alert("Hi");
            }
