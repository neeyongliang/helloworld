4. screen对象

JavaScript 中有几个对象在编程中用处不大,而 screen 对象就是其中之一。涉及移动设备的屏幕大小时,情况有点不一样。运行 iOS 的设备始终会像是把设备竖着拿在手里一样,因此返回的值是 768×1024。而 Android 设备则会相应调用 screen.width 和 screen.height 的值。
