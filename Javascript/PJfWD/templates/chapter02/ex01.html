除了以下两种情况,脚本执行是按照顺序执行的:

  • defer
  • async

以下两种作法相同:

后置(推荐)

<body> <!-- 这里放内容 --> <script type="text/javascript" src="example1.js"></script> <script type="text/javascript" src="example2.js"></script> </body>

延迟

<head> <title>Example HTML Page</title> <script type="text/javascript" defer="defer" src="example1.js"></script> <script type="text/javascript" defer="defer" src="example2.js"></script> </head>

XHTML过渡

在将页面的 MIME 类型指定为 "application/xhtml+xml" 的情况下会触发XHTML 模式。并不是所有浏览器都支持以这种方式提供 XHTML 文档。

<script type="text/javascript"> //<![CDATA[ function compare(a, b) { if (a < b) { alert("A is less than B"); } else if (a > b) { alert("A is greater than B"); } else { alert("A is equal to B"); } } //]]> </script>
