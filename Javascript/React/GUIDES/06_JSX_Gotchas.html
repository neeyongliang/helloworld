HTML实体

JSX会默认转义所有字符.

最简单的方法是使用Unicode编码.确保文件是UTF-8编码且网页也指定为UTF-8.

安全的做法是先找到实体的Unicode编号,然后在JavaScript中字符串里使用.


              
{'First' \u00b7 'Second'}

              
{'First' + String.fromCharCode(183) + 'Second'}

自定义HTML属性

如果网HTML元素里传入HTML规范里不存在的属性, React不会显示它们.如果需要使用自定义属性, 要加'data-'前缀.以'aria-'开头的网络无障碍应用属性可以正常使用.

<\div data-custom-attribute="foo" />
<\div aria-hidden={true} />
